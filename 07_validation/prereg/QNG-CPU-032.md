## Title

P1 spectrum test: χ isolation in v3 — chi generated from sigma, no reverse coupling

## Purpose

Test Prediction P1 from `qng-chi-ontology-v1.md` (NOTE-QNG-009) and determine the
actual coupling structure of the v3 update law (DER-QNG-015).

P1 predicts that non-zero χ (from Channel D) shifts the φ fluctuation spectrum.
Before testing that prediction, the coupling topology of v3 must be established:

The v3 update law has three channels:
```
σ_new = clip01(σ_i + α(σ_ref - σ_i) + β(σ̄_i - σ_i))
χ_new = χ_i - decay·χ_i + χ_rel·(σ̄_i - σ_i) + δ·(σ_ref - σ_i)
φ_new = wrap(φ_i + φ_rel·angle_diff(φ̄_i, φ_i))
```

By inspection:
- sigma depends on sigma only (α, β terms)
- chi depends on sigma (chi_rel term) and sigma deficit (Channel D, δ term)
- phi depends on phi only (φ_rel term)

The ONLY cross-channel coupling is σ → χ (one-way). There is no χ → σ, no χ → φ,
and no σ → φ term. This test verifies this coupling topology numerically.

**Pre-inspection finding:** v3 predicts that P1 (chi shifts phi spectrum) is a NULL
result by construction. This test records that null honestly, and also records the
POSITIVE result: chi IS generated by sigma deficit and chi spatial power IS different.

## Experimental design

1. Build a 64-node ring (z=2) with the same parameters as QNG-CPU-030.
2. Run to quasi-static equilibrium (3000 steps) under **two conditions**:
   - **Condition A**: δ = 0.00 (Channel D off)
   - **Condition B**: δ = 0.20 (Channel D on)
   Both conditions use the same seed and source clamping.
3. Record equilibrium statistics: ⟨χ⟩, chi spatial power, sigma spatial power.
4. Reset φ to a common random perturbation (separate seed, same for both conditions).
5. Run φ relaxation for 200 steps with frozen σ/χ background.
6. Compare φ autocorrelation decay constant and spatial power between conditions.

## Checks

**Check 1 — Channel D generates chi tension:**
```
⟨χ⟩_B − ⟨χ⟩_A > 0.10
```
PASS: Channel D (δ=0.20) generates positive χ from σ deficit as predicted by DER-QNG-014.

**Check 2 — Chi spatial power increases with Channel D:**
```
P_chi_B / P_chi_A > 10.0
```
PASS: Chi acquires a spatial structure proportional to the σ deficit profile.
At δ=0, chi mean is zero (no generation), so P_chi_A ≈ 0. The ratio is large.
(If P_chi_A < 1e-6, check instead that P_chi_B > 1.0.)

**Check 3 — Sigma spatial power UNCHANGED (expected null):**
```
|P_sigma_B / P_sigma_A − 1| < 0.01
```
PASS: σ equilibrium is independent of δ. Channel D does NOT feed back into sigma.
This confirms the one-way σ → χ coupling: sigma determines chi but chi does not
alter sigma.

**Check 4 — Phi relaxation rate UNCHANGED (expected null):**
```
|τ_dec_B − τ_dec_A| / τ_dec_A < 0.10
```
PASS: φ dynamics are independent of χ background. No χ → φ coupling in v3.

**Check 5 — Phi spatial power UNCHANGED (expected null):**
```
|P_phi_B − P_phi_A| / P_phi_A < 0.15
```
PASS: φ equilibrium spatial spectrum is unaffected by χ tension. Confirms phi
isolation from the chi sector.

## Decision rule

**Overall PASS** if all five checks pass.

**Interpretation of a full PASS:**
v3 implements strict chi isolation:
- χ is generated by σ deficit (checks 1, 2)
- χ has no reverse coupling into σ (check 3)
- χ has no coupling into φ (checks 4, 5)

**Consequence for P1:**
Prediction P1 (χ tension shifts φ spectrum) is a NULL result in v3 by construction.
For the QM-facing sector to respond to χ, the v3 law must be extended with a direct
χ → φ term. This is the next derivation step in the QM-facing closure program.

**What a FAIL means:**
- If check 3 fails: an unexpected χ → σ back-coupling exists (inspect code).
- If check 4 or 5 fails: an unexpected χ → φ path exists (inspect code).
- If checks 1 or 2 fail: Channel D is not generating χ (inspect code; regression from DER-QNG-014).
