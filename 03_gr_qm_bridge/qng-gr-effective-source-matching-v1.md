## Motivation

The rebuilt GR lane already has:

- linearized curvature `R_lin`
- geometry-side baseline `K_C`

The bridge lane now also has:

- coherence/transport source `Q_src`
- density-source channel `S_src`

And the QNG core has:

- matter-like proxy `M_eff`

The next natural GR-R3 question is:

- can a disciplined scalar source composite explain the curvature side better than the geometry baseline and bridge-only source models

## Scalar matching model

Use the first Einstein-like scalar matching ansatz:

`R_lin(i) ~ a K_C(i) + b Q_src(i) + c S_src(i) + d M_eff(i)`

where:

- `K_C` is the geometry-side baseline
- `Q_src` is the bridge coherence/transport source
- `S_src` is the density-source channel from the QM generator
- `M_eff` is the matter-like proxy

## Why this matters

This is not yet an Einstein equation.

But it is the first place where the rebuilt GR lane can ask a field-equation-shaped question:

- can curvature be matched better by a structured source composite than by geometry alone

This is the correct scalar precursor to any later tensorial recovery program.

## Minimal success condition

The first scalar source matching is useful if:

1. the four-channel fit improves over the geometry-only, v1, and v2 source fits
2. the added matter coefficient is nonzero
3. the fitted relation retains history imprint

## Interpretation

If this works, then the rebuilt GR lane is no longer only:

- a first effective source composite matching step
