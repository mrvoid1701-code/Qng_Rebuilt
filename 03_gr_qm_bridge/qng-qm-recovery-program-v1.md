## Scope

This file does not claim exact QM recovery.

It defines the staged recovery ladder that the rebuild must follow if such a claim is ever to be justified.

## Recovery ladder

### Stage QM-R1: state and source-augmented continuity assembly

Assemble a complex state candidate and a current-like candidate that satisfy a controlled continuity-style relation, possibly with an explicit source term from the native QNG load sector.

This is the first place where the rebuilt proxy sector must start behaving like a genuine dynamical quantum structure.

### Stage QM-R2: local generator assembly

Show that the state candidate admits a controlled generator-like description:

- phase rotation
- amplitude flow
- and neighborhood transport

without arbitrary per-step redefinition.

### Stage QM-R3: operator-level candidate

Show that observables can be represented as stable operator-like objects acting on the recovered state space.

### Stage QM-R4: spectrum and mode structure

Show that the operator/generator sector admits a meaningful spectrum, mode language, and propagator interpretation.

### Stage QM-R5: exact recovery claim

Only after QM-R1 through QM-R4 can the rebuild honestly ask whether:

- canonical commutator structure
- Schrödinger-like or field-theoretic evolution
- and exact QM observables

are recovered or deformed in a controlled way.
