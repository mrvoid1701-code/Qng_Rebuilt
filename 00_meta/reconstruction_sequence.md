# Reconstruction Sequence

## Stage 1: GR pure

Build the minimal GR spine first.

Targets:

- metric objects
- curvature objects
- Einstein equations
- action and variation
- classical limits and sanity identities

Output condition:

GR folder can stand on its own without any QNG symbol.

## Stage 2: QM pure

Build the minimal QM spine second.

Targets:

- state space
- observables
- commutator structure
- propagators
- thermal/semiclassical preliminaries

Output condition:

QM folder can stand on its own without any QNG symbol.

## Stage 3: GR-QM bridge

Build only the minimal bridge layer.

Targets:

- correspondence objects
- semiclassical interface
- shared consistency limits
- discrete/continuum translation notes

Output condition:

Bridge layer references GR and QM, but still does not require native QNG ontology.

## Stage 4: QNG pure

Now define the native theory.

Targets:

- primitive ontology
- native update law
- native action or constraints
- emergent metric or geometry map
- native time/memory variables

Output condition:

QNG can be stated internally before talking about data.

## Stage 5: Phenomenology

Only after QNG pure is coherent.

Targets:

- trajectory
- lensing
- rotation
- cosmology
- timing

Output condition:

Each domain is an application layer, not a source of ontology.

## Stage 6: Validation and governance

Last, not first.

Targets:

- prereg structure
- CPU/GPU test lanes
- manifests
- freeze decisions
- switch notes

Output condition:

Validation measures the theory; it does not define it.
