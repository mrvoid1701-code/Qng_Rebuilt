# GR Pure

This folder contains GR-only material.

Allowed:

- metric structure
- curvature
- Einstein equations
- geodesics
- weak-field and PPN objects
- action principles

Forbidden:

- QNG-native primitives such as `Sigma`, `chi`, `tau`
- dataset fitting
- governance logic

- `gr-primitives.md`
