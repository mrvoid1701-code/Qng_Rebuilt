# Core Manual v1

Type: `note`
ID: `NOTE-GOV-008`
Status: `draft`
Author: `C.D Gabriel`

## Objective

State the provisional canonical form of the rebuilt QNG backbone in one place.

## Scope

This manual describes only the current provisional core.

It does not freeze:

- exact GR recovery
- exact QM recovery
- exact back-reaction closure
- final matter ontology
- real-data interpretation

## Core statement

The rebuilt theory is currently best formulated as:

- a relational substrate
- with local history-sensitive updates
- whose coarse-grained behavior yields a split effective layer `(C_eff, L_eff)`
- from which geometry, bridge sectors, and phenomenology descend

This is the strongest clean statement currently supported by the rebuild.

## Level structure

The rebuilt core uses four levels.

### Level 1: substrate ontology

Current best provisional ontology:

- dynamic relational substrate `G(t)`
- local node state container `N_i(t)`
- local update operator `U`

The detailed primitive status of internal node coordinates remains open.

### Level 2: native dynamics

The native theory is defined by:

`N_i(t+1) = U_local(N_i(t), neighborhood data, history data, local inputs)`

The current irreducible content is:

- locality
- discreteness
- history sensitivity

This is the true dynamical spine of the rebuild.

### Level 3: effective layer

The current effective layer is explicitly split:

- `C_eff`
  coherence / compatibility content
- `L_eff`
  memory-load content

This is a deliberate correction of the old repo, where one scalar carried too many incompatible roles.

### Level 4: derived sectors

From the effective layer, the rebuild currently derives:

- geometry proxies from `C_eff`
- GR-facing weak-field proxies
- QM-facing coherence / generator / spectrum proxies
- bridge closure and source-response proxies
- phenomenology proxies

## Geometry statement

The current geometry claim is not:

- "`Sigma` is primitive"

but:

- a coherence-sensitive effective field can generate a stable geometry proxy

The current geometric path is:

`C_eff -> geometry proxy -> Lorentzian signature proxy -> weak-field assembly -> linearized curvature proxy`

This is currently the strongest recovery lane in the rebuild.

## QM statement

The current QM claim is not:

- exact quantization has been recovered

but:

- the rebuilt state candidate supports coherent local complex update structure and a meaningful mode/spectrum proxy

The current QM path is:

`(C_eff, phi) -> correlator proxy -> local generator proxy -> mode/spectrum proxy`

The weak point remains:

- continuity-style balance

So the rebuilt QM core currently looks more generator-like than continuity-like.

## Bridge statement

The current bridge claim is:

- the same native dynamics feeds a GR-facing sector and a QM-facing sector through different reductions

and also:

- a first back-reaction proxy closure exists
- a first source-response consistency step exists

This is enough to justify a live bridge architecture, but not a final closure theorem.

## Matter statement

The rebuilt theory currently allows:

- a first matter-like proxy `M_eff`

built from:

- memory-load
- coherence defect
- transport activity

This is useful because it separates matter-like content from pure geometry more cleanly than the old repo.

It is not yet a final matter ontology.

## Phenomenology statement

All current phenomenology is downstream only.

The rebuilt phenomenology layer currently includes:

- trajectory lag proxy
- lensing proxy
- rotation support proxy
- timing delay proxy
- cosmology expansion proxy

These are application layers, not sources of ontology.

## What is provisionally frozen

The following may now be treated as provisional backbone:

- native memory-sensitive update
- split effective layer `(C_eff, L_eff)`
- geometry proxy path
- GR weak-field path
- QM generator / spectrum path
- back-reaction proxy closure
- source-response consistency
- matter-like proxy

## What remains explicitly open

The following remain outside the provisional freeze:

- exact GR theorem
- exact QM theorem
- exact back-reaction law
- final Lorentzian recovery
- final matter interpretation
- final status of `Sigma`
- final status of `chi`
- any universal `tau`

## Practical reading rule

If a new derivation contradicts this manual, then one of two things must happen explicitly:

1. the derivation stays experimental
2. the manual is revised on purpose

Nothing should silently rewrite the backbone.
