# QNG Pure

This folder contains the native QNG theory after GR and QM foundations are separated.

Allowed:

- primitive ontology
- node/edge/state objects
- native update laws
- memory variables
- emergent geometry maps
- native actions and constraints

Forbidden:

- direct data fitting as theory definition
- governance notes

- `qng-ontology.md`
- `qng-primitives.md`
- `qng-update-law.md`
- `qng-emergence-map.md`
