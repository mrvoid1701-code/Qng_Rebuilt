## Objective

List the questions QNG must answer after GR, QM, and the bridge are already separated.

## Core questions

1. What is the native substrate?
2. What are the primitive local state variables?
3. What is the update law?
4. What is the memory structure?
5. How is geometry emergent rather than assumed?
6. How does the quantum layer appear without being inserted by hand?
7. Which effective observables are truly predictions rather than definitions?

## Anti-confusion rule

If a question can already be answered completely inside:

- `GR pure`
- `QM pure`
- or `GR-QM bridge`

then it is not a uniquely QNG question.

## Immediate use

This file will act as a checklist when we start writing the first real QNG-native derivations.
